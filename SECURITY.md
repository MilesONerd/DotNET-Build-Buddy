# Security Policy for DotNET Build Buddy

## Reporting a Vulnerability

If you discover a security vulnerability in DotNET Build Buddy, please report it **privately** via email to:

**security@milesonerd.8shield.net** 

Include as much detail as possible, such as:

- Steps to reproduce the issue
- Expected and actual behavior
- Any relevant screenshots or logs

---

## Response Time

We aim to respond within **5 business days** of receiving a report. 

---

## Responsible Disclosure

Once the vulnerability is fixed, we allow the reporter to publicly disclose the issue in a responsible manner.  
Please coordinate with us before sharing details publicly to ensure users are protected.

---

## Scope

This policy applies to DotNET Build Buddy, including:

- Source code in this repository
- All releases and binaries
- GitHub workflows and automation scripts
